SQL queries for Beancount

This is a list of example SQL queries for Beancount. It should be a reference and inspiration for users of beancount. If you want to contribute and add your own query to this list please create an issue or submit a pull request here: github.com/aumayr/beancount-sql-queries. For more information about the SQL-like query language for beancount visit the Beancount Query Language documentation.
